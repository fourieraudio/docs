# Antares

How to install **Antares** plugins on your **transform**.engine!

## Installation

**Plugin Download Link:** Please contact [support@antarestech.com](mailto:support@antarestech.com) to request an offline installer.

**Licensing Method:** iLok ([Licensing Guide](../ilok.md))

**Plugin Format:** Windows VST3

**Installer File Format:** `.exe`

1. After following the [Accessing Windows steps](../installation.md#accessing-the-plugin-host-to-install-plugins), you will be in a position to install plugins.
2. On your personal device, copy the downloaded offline installer to a USB stick.
3. Plug it into any of the USB ports on the **transform**.engine.
4. In the search bar, type ‘This PC’, it will find the ‘This PC’ app and hit enter.
5. Locate your USB stick in the window and find your Antares installer file.
6. Run the Application (`.exe` file) for the desired plugin and follow the installer steps.
7. Repeat the process if you have further plugins to install.
